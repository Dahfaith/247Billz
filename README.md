# 247Billz

247Billz is a modern, intuitive, and secure invoicing and quotation platform designed to help businesses manage their billing effortlessly.

## Features

- **Professional Invoices & Quotations:** Generate sleek, branded documents in seconds.
- **Multi-Currency Support:** Bill your clients in their preferred currency.
- **Automated Payments:** Seamless integration with Flutterwave for direct card, bank, and mobile money payments.
- **Client CRM:** Keep track of all your active clients and their billing history.
- **Dashboard Analytics:** Monitor your revenue, outstanding payments, and recent activity at a glance.
- **PDF Generation:** Instantly download any invoice or estimate as a professional PDF.

## Tech Stack

- **Frontend & API:** Next.js, React, Tailwind CSS
- **Database:** Supabase (PostgreSQL)
- **Payments:** Flutterwave API
- **Emails:** Resend API

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Copy `.env.example` to `.env.local` and fill in your Supabase and Flutterwave keys
4. Run the development server with `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)
